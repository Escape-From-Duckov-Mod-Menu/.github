**Installation Guide**  
1. Download the latest archive.  
2. Extract to a dedicated folder.  
3. Run `DuckovModMenu.exe` as administrator.  
4. Launch Escape From Duckov.  
5. Press INSERT to open the mod menu.  
6. Navigate tabs and apply real-time changes.  

Re-attach after game restarts or new raids.
